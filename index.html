What This Course Covers

Students develop foundational skills in deep learning and generative AI applied to software engineering, from mining software repositories and training language models to prompting LLMs and detecting hallucinations in generated code.

mining repositories language models evaluation metrics deep learning prompting code hallucinations code generation vibe coding
⚡

Build with AI

Go from zero to shipping real applications using LLMs as your coding partner. Learn what works, what doesn't, and why.

🧪

Understand the Science

Explore how language models learn from code: tokenization, probability, attention, and the training pipeline behind tools like Copilot.

🔍

Think Critically

Evaluate AI-generated code rigorously. Measure quality, detect hallucinations, and understand when to trust — and when not to.
